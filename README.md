## 소개
* 개발 환경 : Windows / Eclipse / Apache Tomcat
* 개발 언어 : JAVA / HTML / CSS
* 참여인원 : 3명

## 담당 기능
* 게시판 글쓰기(Art, Lecture)
    * 네이버 에디터 API 사용
    * 다음지도 API 사용
* 자유게시판, QnA 게시판 CRUD 구현
* IMPORT / 카카오페이 API 사용하여 결제 기능 구현
* 계층형 쿼리 구조로 댓글 기능 구현
